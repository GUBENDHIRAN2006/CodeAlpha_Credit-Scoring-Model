# CodeAlpha_Credit-Scoring-Model
A machine learning model that predicts whether a loan application is Approved or Rejected based on applicant financial data. Built using Random Forest Classifier with feature scaling and encoding for both numerical and categorical inputs. Trained and evaluated with accuracy, precision, recall, and F1-score metrics. 

💳 Credit Card Loan Approval Prediction
A machine learning model that predicts whether a loan application will be Approved or Rejected based on applicant financial data.
📊 Features Used
Numerical: Age, Annual Income, Credit Score, Loan Amount, Monthly Income, Net Worth
Categorical: Employment Status, Marital Status
⚙️ Tech Stack
Python | Scikit-learn | Pandas | NumPy | Matplotlib
🔧 Approach

Data preprocessing & feature encoding (OneHotEncoder + StandardScaler)
Model: Random Forest Classifier (200 estimators, max depth 10)
Evaluation: Accuracy, Precision, Recall, F1-Score, Confusion Matrix
Interactive user input system for live predictions

🚀 How to Run
bashpip install pandas numpy scikit-learn matplotlib
jupyter notebook Credit_limit.ipynb
📁 Dataset
Place your Loan new datset.csv file in the project directory before running.
